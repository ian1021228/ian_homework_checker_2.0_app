📖 系統功能與使用教學

⬇️ 安裝為 App (PWA)

只要點擊右上角的「安裝 App」，系統就會在您的手機桌面或電腦上建立一個專屬圖示。打開後不僅沒有網址列，全螢幕的體驗更是快又流暢！

💾 資料安全與本地儲存

本系統採用最新的 File System Access API，所有的資料都會直接寫入您設定的 .json 檔案中，完全無雲端外洩風險！系統會在每次操作時自動覆蓋儲存，讓您無後顧之憂。

📝 作業點收與管理

  • 智慧缺號排除：支援單號(例如 5)與範圍(例如 20-25)輸入，自動跳過該號碼。
  • 全班一鍵變更：點進作業後，可使用上方「⚡ 全班一鍵變更」下拉選單，瞬間將全班狀態改為「已繳交」或「已完成」。
  • 點擊狀態切換：滑鼠點擊或手指觸控座號方格，即可在不同狀態間循環切換。

歡迎使用作業點收系統 🎉

本系統採用最新的本地儲存技術，無需註冊或連線雲端資料庫。

建議您建立一個專屬的 .json 檔案來保存資料，系統會自動幫您讀取與覆蓋更新，不怕資料遺失！
