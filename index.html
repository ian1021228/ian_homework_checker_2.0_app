作業點收系統 - 終極功能指南

📱 安裝與跨平台體驗 (PWA)

點擊首頁右上角的「安裝 App」即可將系統加至主畫面。安裝後將移除瀏覽器網址列，為您提供無干擾的全螢幕沉浸式體驗，並且完整支援離線操作模式。

🔐 隱私與硬碟同步技術

本系統導入最先進的 File System Access API。只要綁定本機的 JSON 檔案，所有的操作變更都會即時覆寫至您的硬碟中，實現 0 雲端外洩、100% 隱私安全的本機端資料庫架構。

📝 智慧作業點收與防呆

  • 區間缺號解析：支援「單號」與「範圍」混合輸入（例：4, 15-20），系統將自動為您生成完美的座號矩陣，並自動記憶該班級配置。
  • 全班一鍵變更：提供下拉選單，能瞬間將全班狀態統一改為「已繳交」或「已完成」。
  • 果凍微動效：點擊座號方格即會觸發立體軟膠回彈特效與光暈，色彩切換絲滑無閃爍。

🎨 作業種類自訂與美學標籤

在「作業種類」設定中，您可以自由建立各科專用的點收類別（至多配置 4 種核心狀態切換）。點擊色點即可更換高級標籤配色，並能自訂哪種狀態才算作「完成」。

🏫 擬真黑板聯絡簿與行距調節

  • 視覺與版面自訂：極致烤漆黑板與粉筆光澤，同時支援 A+/A- (字體縮放) 以及 ↕️+/↕️- (行距縮放)，確保在各式投影螢幕上皆具完美閱讀性，且系統會自動記憶您的喜好。
  • 連動整合：滑鼠懸停於聯絡事項，點擊「📖」按鈕，即可將該行文字瞬間無縫轉發為今日的作業點收清單。

📊 學生詳情與全勤分析

「學生詳情」如同您的控制儀表板，將彙整該班級「所有未完成」的作業與狀態，並提供🏆 全勤榮譽名單，讓表現優異的學生一目瞭然。

⏳ 時光機與自動防呆救援

在「資料管理」中配備了強大的時光機功能。系統於每日背景輪替保留前三天的快取紀錄，若不慎誤刪班級或作業，點擊還原即可瞬間救回。您也能手動匯出 JSON 備份檔至雲端封存。

歡迎使用作業點收系統 🎉

本系統採用最新的本地儲存技術，無需註冊或連線雲端資料庫。

建議您建立一個專屬的 .json 檔案來保存資料，系統會自動讀取與覆蓋更新！
